django编码问题

将django默认编码设置为utf-8，进入django/conf/global_setting.py设置

DEFAULT_CHARSET = ‘utf-8’

还有就是在写代码的时候里面如果出现中文，在run的时候会报错，这里需要将setting里的设置改一下

LANGUAGE_CODE = ‘zh-cn’

Request

Request里面只记得几个常用的属性。
Request.method == “POST” or Request.method == “GET”,url请求方式。get 和 post
file = Request.FILES[“key”] 获得名为key的文件，只用过这么多
file.read()是获得所有数据
当上传数据过大的时候可以用file.multiple_chunks(),不过我基本就用过file.chunks()
其他的属性可以去看官方文档，我比较偷懒直接找了个中文的。
